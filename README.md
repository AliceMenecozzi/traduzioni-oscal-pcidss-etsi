# traduzioni in OSCAL di PCI-DSS v4.0.1 e ETSI EN 303 645 v.3.1.3
Questo repository contiene i file di traduzione in formato OSCAL (YAML, JSON, XML) realizzati come parte della tesi di laurea triennale in Ingegneria Informatica

Le traduzioni rappresentano la trasposizione strutturata dei requisiti di due standard di sicurezza informatica secondo il framework **Open Security Controls Assessment Language (OSCAL)** definito dal *National Institute of Standards and Technology (NIST)*.

## Struttura del repository
- **/pci-dss/** → File di traduzione dello standard *PCI-DSS v4.0.1*  
  - `pci-dss_catalog_v3.yaml`  
  - `pci-dss_catalog_v3.json`  
  - `pci-dss_catalog_v3_formattato.xml`
- **/etsi/** → File di traduzione dello standard *ETSI EN 303 645 v3.1.3*  
  - `etsi en 303 645_v2.yaml`  
  - `etsi en 303 645_v2.json`  
  - `etsi en 303 645_v2_formattato.xml`
 
## Note
- Tutti i file sono conformi agli schemi OSCAL versione 1.1.3
- La validazione e la conversione tra formati sono state eseguite con lo strumento [oscal-cli] come descritto nel documento della tesi.
